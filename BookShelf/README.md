# Green Bookshelf (React SPA)

A simple green-and-white book catalog with:
- Persistent header (shared on all pages)
- Client-side routing (Home / Login / Register / About)
- Mock data stored in parent component state (10 books)
- Combined search + sort (and a simple “Featured only” mode)
- Responsive layout (mobile + desktop)

## Run locally (VS Code)
1) npm install
2) npm start

## Deploy on Render (Static Site)
- Build command: `npm install && npm run build`
- Publish directory: `build`
